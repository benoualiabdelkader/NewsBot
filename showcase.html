Integrated APIs

This application demonstrates integration with multiple news and media APIs:

  • NewsAPI.org: For fetching top headlines and category-based news articles
  • Google Custom Search API: For comprehensive web search capabilities
  • YouTube Data API: For retrieving live news streams and related videos
  • Google Gemini AI: For powering the intelligent news assistant chatbot

Note: The demo currently uses mock data for API responses to ensure reliable demonstration. In a production environment, these would be replaced with live API calls.

Technical Implementation

This project is built using the following technologies:

  • Frontend: HTML5, CSS3, JavaScript (ES6+)
  • Backend: Node.js with Express
  • AI Integration: Google Gemini API
  • Responsive Design: Custom CSS with Flexbox and Grid layouts
  • Data Storage: LocalStorage for favorites and user preferences
